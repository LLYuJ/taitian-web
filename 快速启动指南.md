# 泰田集团官网 - 快速启动指南

## 项目概述

这是一个现代化的企业官网项目，采用前后端分离架构：
- **前端**: Vue 3 + Vite + Element Plus（现代工业风格，白色 + #2CB5BE 配色）
- **后端**: FastAPI + SQLAlchemy + SQLite（使用uv包管理）
- **功能**: 企业展示网站 + 管理员后台 + 媒体文件管理

## 环境要求

### 后端
- Python 3.10+
- uv 包管理器

### 前端
- Node.js 16+
- npm 或 yarn

## 安装步骤

### 1. 安装 uv 包管理器（后端）

```bash
# macOS/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows（PowerShell）
irm https://astral.sh/uv/install.ps1 | iex
```

### 2. 安装 Node.js（前端）

从 [nodejs.org](https://nodejs.org/) 下载并安装最新的 LTS 版本。

## 启动项目

### 方式一：使用启动脚本（推荐）

#### macOS/Linux

```bash
# 启动后端（终端1）
chmod +x start-backend.sh
./start-backend.sh

# 启动前端（终端2）
chmod +x start-frontend.sh
./start-frontend.sh
```

#### Windows

使用 PowerShell 或 Git Bash：

```bash
# 启动后端（终端1）
cd backend
uv pip install -e .
uv run uvicorn app.main:app --reload

# 启动前端（终端2）
cd frontend
npm install
npm run dev
```

### 方式二：手动启动

#### 启动后端

```bash
cd backend

# 安装依赖
uv pip install -e .

# 启动服务
uv run uvicorn app.main:app --reload
```

后端将在 http://localhost:8000 运行
- API 文档：http://localhost:8000/docs
- 健康检查：http://localhost:8000/health

#### 启动前端

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

前端将在 http://localhost:5173 运行

## 生产部署

### 后端部署

```bash
cd backend

# 安装依赖
uv pip install -e .

# 使用生产服务器（例如 gunicorn）
uv run gunicorn app.main:app -w 4 -k uvicorn.workers.UvicornWorker --bind 0.0.0.0:8000
```

### 前端部署

```bash
cd frontend

# 构建生产版本
npm run build

# dist 目录包含构建后的静态文件
# 可以部署到 Nginx、Apache 等 Web 服务器
```

### 环境变量配置

复制 `backend/.env.example` 为 `backend/.env` 并修改配置：

```env
# 数据库
DATABASE_URL=sqlite+aiosqlite:///./taitian.db

# JWT密钥（生产环境必须修改）
SECRET_KEY=your-secret-key-here

# CORS
ALLOWED_ORIGINS=https://yourdomain.com
```

## 常见问题

### 1. uv 命令未找到
确保已正确安装 uv 包管理器，并重启终端。

### 2. 端口被占用
修改后端端口：在启动命令中添加 `--port 其他端口号`
修改前端端口：在 `vite.config.js` 中修改 `server.port`

### 3. CORS 错误
检查后端 `app/core/config.py` 中的 `ALLOWED_ORIGINS` 配置。

### 4. 数据库初始化失败
删除 `backend/taitian.db` 文件，重新启动后端自动创建。
